# Contributing to MarinMC Launcher

Thank you for your interest in contributing to the MarinMC Launcher! We welcome contributions from the community.

## Code of Conduct

By participating in this project, you agree to maintain a respectful, welcoming, and inclusive environment.

## How Can I Contribute?

### Reporting Bugs
- Use the provided **Bug Report** template.
- Check if the bug has already been reported before opening a new issue.
- Include as many details as possible, including launcher logs and steps to reproduce.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Describe the feature clearly and why it would be beneficial to players.

### Pull Requests
1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/amazing-feature`.
3. Keep code clean and write TypeScript types.
4. Ensure files are linted and compile correctly: `npm run build`.
5. Commit your changes with clear messages.
6. Push to the branch and open a Pull Request.

## Coding Style
- Use standard TypeScript types and interfaces.
- Keep components small, modular, and reusable.
- Follow Tailwind CSS patterns for styling.
